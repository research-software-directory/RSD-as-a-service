# Frontend Default Settings Configuration

Edit the file on the Path:
`frontend/public/data/settings.json`

## Enabling feedback button in the frontend

```json
{
  "feedback": {
    "enabled": true,
    "url": "https://feedback.example.com",
    "issues_page_url": ""
  }
}
```
